# 邮轮舱房选择 · Claude Code 插件

探索不同邮轮舱房类型及其价格区间，帮助用户根据预算选择最合适的舱房配置。非常适合计划家庭旅行或团体出行的用户。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/craftwave-skill-5-claude-plugin
/plugin install craftwave-skill-5@craftwave-skill-5-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **邮轮舱房选择**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
